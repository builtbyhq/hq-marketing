# hq-www

### Getting Started
1. Install Node.js using [NVM](https://github.com/creationix/nvm) or by going to [Nodejs.org](https://nodejs.org)
2. ```npm install -g zab```
3. ```zab build```
4. ```zab watch``` or ```zab server```
5. ```zab --help``` for more info
6. Visit us at [zab.io](https://zab.io)
